# Ausführung:

### Prerequisites:
* ```node```
### Zuerst
* ```npm install``` - Installiert die Node Packages.
### POC Ausführen
* ```npm run poc``` - Führt den poc aus.
### FAKE_API Ausführen
* ```npm run fakeapi``` - Führt den api aus.
### API Ausführen
* ```npm run api``` - Führt den api aus.
